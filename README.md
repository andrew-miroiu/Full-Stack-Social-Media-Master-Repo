# Social Media Platform – Repository Wrapper

This repository serves as an overview and entry point for my social media platform project.

The platform consists of:
- A *React + TypeScript* frontend  
- Two backend services: one implemented in *Node.js, the other in **Spring Boot (Java)*  
- Both backends use *Supabase* for authentication and database access  
- All services connect to the same Supabase database

---

## 📦 Repositories

### *1. Node.js Backend (TypeScript)*
A backend service built with Node.js and TypeScript, using Supabase for auth and data management.

### *2. Spring Boot Backend*

## 🏗️ Architecture Summary

- *Frontend:* React + TypeScript  
- *Backend Services:*  
  - Node.js (TypeScript)  
  - Spring Boot (Java)  
- *Database & Authentication:* Supabase, PostgreSQL, JWT
- Both backends interact with the same Supabase project and database schema.

This repository serves purely as an index for navigating between the separate backend implementations.
